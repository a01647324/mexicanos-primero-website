# Mi Escuela Primero 

> A web platform by students of Tecnológico de Monterrey in the 4th semester for the formative partner **Mexicanos Primero Jalisco**,that connects real needs from low-resource schools with donors and organizations interested in helping.

---

## About the Project

**Mi Escuela Primero** is a donation management platform that allows visitors to explore a dynamic catalog of school needs from schools around Jalisco in vulnerable contexts. Donors will be able to filter needs by municipality, category, and school, add items to a donation cart, and submit a formal donation request. The platform also includes a full administrative panel for staff members to manage schools, needs, requests, and users.

## User Roles

| Role | Access |
|---|---|
|`browsing user` | Access to needs of the organization, can't submit donation requests |
| `donador` | Submit donation requests, view personal profile and history |
| `admin` | Full access to all administrative panel features and write operations |
| `lector` | Read-only access to the administrative panel |

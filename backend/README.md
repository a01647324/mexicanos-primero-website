## Requisitos
- Contraseña de la base de datos (me la piden)

## Instalación
1. Clona el repo y entra a la carpeta backend en consola con el siguiente comando:
   cd backend

   Nota: Eliminen el proyecto de sus computadoras y vuelvanlo a clonar para que se configure al 100%

2. Instala dependencias en la carpeta backend:
   npm install

3. Copia el archivo .env.example
   Renombralo como .env
   Edita la DATABASE_URL con la contraseña real 

4. Corre el servidor con el siguiente comando en la carpeta backend:
   npm start 

   O con node server.js

El servidor queda en http://localhost:3000
Prueba: http://localhost:3000/api/test
